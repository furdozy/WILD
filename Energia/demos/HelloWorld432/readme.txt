Before running follow these steps:

1. Make sure that the MSP432p401r drivers are installed and the device is selected.

2. Copy the LiquidCrystal.h file from the hardware\MSP430 directory to the main library folder

3. Change the second definition of P2_5 in the MSP432 header to P2_6. There should be a comment indicating that this pin is meant to be defined as P2_6
